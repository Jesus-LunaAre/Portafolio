# Imágenes de Empresas

Coloca aquí las imágenes/logos de las empresas en formato PNG o JPG.

## Estructura de archivos esperada:

- `ypf.png` - Logo de Yacimientos Petrolíferos Fiscales
- `tgn.png` - Logo de Transportadora Gas del Norte
- `pmi.png` - Logo de Philip Morris International
- `kimberly.png` - Logo de Kimberly Clark
- `dapsa.png` - Logo de DAPSA
- `oxxo.png` - Logo de OXXO
- `heineken.png` - Logo de Heineken
- `femsa.png` - Logo de Femsa
- `xpertal.png` - Logo de Xpertal
- `ragasa.png` - Logo de Ragasa

## Recomendaciones:

- Tamaño recomendado: 200x200px o más
- Formato: PNG con fondo transparente (preferible) o JPG
- Si una imagen no se encuentra, se mostrará un emoji como respaldo
