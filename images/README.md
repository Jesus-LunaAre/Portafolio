# Imágenes del Portafolio

## Imagen de Perfil

Coloca tu foto de perfil aquí con el nombre:
- `profile.jpg` o `profile.png`

### Recomendaciones:
- Tamaño recomendado: 400x400px o más (se ajustará automáticamente)
- Formato: JPG o PNG
- Fondo: Preferiblemente con fondo transparente o sólido
- La imagen se mostrará en formato circular

### Si no hay imagen:
Si no colocas una imagen, se mostrará un círculo con tus iniciales "JL" como respaldo.
